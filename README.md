# Simple Interest Calculator

## Description

A bash script to calculate simple interest for micro-finance applications. This project supports our mission to empower and provide financial opportunities to low-income individuals.

## Formula

Simple Interest (SI) = (Principal × Rate × Time) / 100

Where:
- **Principal (P)**: Initial amount of money
- **Rate (R)**: Annual interest rate (percentage)
- **Time (T)**: Time period in years

## Prerequisites

- Bash shell
- `bc` calculator (usually pre-installed on Unix/Linux systems)

## How to Use

1. Clone this repository:
```bash
   git clone https://github.com/YOUR-USERNAME/simple-interest-calculator.git
```

2. Navigate to the directory:
```bash
   cd simple-interest-calculator
```

3. Make the script executable:
```bash
   chmod +x simple-interest.sh
```

4. Run the script:
```bash
   ./simple-interest.sh
```

5. Enter the required values when prompted

## Example Output
```
Enter the principal amount:
1000
Enter the rate of interest per annum:
5
Enter the time period in years:
2
Simple Interest = 100.00
```
